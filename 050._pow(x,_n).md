Recursive:
```
class Solution:
    def myPow(self, x, n):
        if not n:
            return 1
        if n < 0:
            return 1 / self.myPow(x, -n)
        if n % 2:
            return x * self.myPow(x, n-1)
        return self.myPow(x*x, n/2)
```

Iterative:
```python 
   def myPow(self, x, n):
        m = -n if n < 0 else n
        y = 1
        while m:
            if m & 1:
                y *= x
            x *= x
            m >>= 1
        return y if n >= 0 else 1/y
```        
