###8. String to Integer (atoi)

题目:
<https://leetcode.com/problems/string-to-integer-atoi/>


难度:
Easy


需要考虑比较多的边界条件&特殊情况


```python
class Solution:
    def myAtoi(self, str):
        """
        :type str: str
        :rtype: int
        """
        
        try:
            str=str.strip().split()[0]
        except IndexError:
            return 0
        if not str[0].isdigit() and str[0]!='-' and str[0]!='+':
            return 0
        result=0
        if str[0]=='-':
            for s in str[1:]:
                if not s.isdigit():
                    break
                result=10*result+int(s)
            result*=-1
        elif str[0]=='+':
            for s in str[1:]:
                if not s.isdigit():
                    break
                result=10*result+int(s)
        else:
            for s in str:
                if not s.isdigit():
                    break
                result=10*result+int(s)
                
        #check
        if result>2**31-1:
            return 2**31-1
        elif result<-2**31:
            return -2**31
        else:
            return result
```
